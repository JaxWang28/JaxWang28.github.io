创建 Bucket

配置 Bucket

桶创建完毕之后，我们需要配置 Bucket，主要是添加自定义域名和创建令牌。前者用来公共访问我们 Bucket 中的文件，后者用来帮助我们上传到 Bucket。

配置自定义域名：设置 -> 公开访问 -> 连接域

这里有 3 点要讲的：

  • 自定义是用来公开访问我们的桶的
  • 我们可以购买域名，然后在 Cloudflare 中注册管理
  • 注册管理域名之后，可以在域中添加已经注册的域或子域

至于如何购买域名，在 CloudFlare 中注册可以 google it，或许我后面会出一期教程
