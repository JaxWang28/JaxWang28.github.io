Linux - 内存管理 - 页描述符

2025/04/29   次阅读量

Linux 使用分页技术，因此对物理内存管理的基本单位是 Page Frame 页框。Linux 必须对每一个 Page Frame 了如执掌，其通过为每个页框分配一个页描述符 struct page 保存页框的基本信息。

对于一个物理地址空间 4GB PAGE_SIZE=4KB Linux 系统，Linux 分配 4GB/4KB = 1M = 2^20 大约一百万个 struct page 结构体，尽管这是个庞大的数量，但这是不可避免的。

Linux 中定义了两个宏进行 PFN 与 struct page 之间的转换：

page_to_pfn
pfn_to_page
